# Spotify Catalog Lookup

This is iOS app that uses the Spotify API to retrieve data about music listed on Spotify

## How to install

1. fork the project or download the .zip
2. open project in XCode v15 or later (earlier versions may or may not compile correctly)
3. run in simulator
